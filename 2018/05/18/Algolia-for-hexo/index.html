为Hexo添加Algolia站内搜索

发表于 2018-05-18 | 分类于 学习

注册Algolia，创建Index

前往Algolia注册页面，注册一个新账户。可以使用GitHub或者Google账户直接登录，注册后的14天内拥有所有功能（包括收费类别的）。之后若未续费会自动降级为免费账户，免费账户总共有10,000条记录，每月有100,000的可以操作数。注册完成后，创建一个新的Index，这个 Index 将在后面使用。
